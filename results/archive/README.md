# Archived Research Outputs

This folder holds historical experiment outputs that are preserved for reference and reproducibility but are not part of the current promoted TradeOS decision surface.

Use this area for:

- superseded sweep outputs
- older comparison CSVs
- older compare-suite directories
- earlier phase-labeled experiment bundles
- archived walk-forward summary artifacts
- preserved historical research evidence

Do not move large sets of files into this folder automatically. Prefer reviewed, manual archival so references and context are not lost.
